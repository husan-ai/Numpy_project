# NumPy Practice (Numpy_project)

This repository contains practical exercises completed using the NumPy library. It focuses on working with arrays, mathematical operations, and data manipulation.

## About
This project was created to strengthen my understanding of NumPy through hands-on practice. It includes 30+ exercises ranging from beginner to intermediate level.

## Topics Covered

- Installing and checking NumPy configuration
- Creating arrays (array, zeros, arange)
- Working with array size and memory
- Mathematical operations and functions
- Vector and matrix operations
- Random number generation
- Array manipulation (reverse, reshape)
- Finding min and max values
- Matrix multiplication (np.dot)
- Comparison operations (>, <, ==)
- Finding common elements (np.intersect1d)
- Sorting arrays
- Working with dates and time (datetime64)
- Advanced NumPy functions

## Technologies
- Python  
- NumPy  

## Workflow
- Each task is implemented separately
- Solutions are written using NumPy functions
- Results are tested and verified

## Purpose
The main goal of this repository is to build a strong foundation in NumPy and prepare for Machine Learning and Data Science tasks.
